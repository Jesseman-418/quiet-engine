# The Listing Engine

AI engine for top-producing real estate agents. Resurrects dormant CRM leads, scores them HOT/WARM/COLD, drafts personalized re-engagement in your voice. Installed in 7 days.

Live: [quiet-engine.vercel.app](https://quiet-engine.vercel.app/) *(rename pending)*

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Geist Sans + Geist Mono + Instrument Serif

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Manual deploy only (per AIOS policy):

```bash
npx vercel --prod
```
